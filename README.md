# AgendaGuru - Manajemen Kelas Digital

Classroom management application for attendance, grading, scheduling, and journals.

## Features
- Multi-user authentication
- Attendance tracking
- Grade sheets
- Class scheduling
- Daily journal/agenda
- Integrated with Turso DB (SQLite in the cloud)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Configure environment:
   Create a `.env` file based on `.env.example` and add your Turso URL and Auth Token.
3. Run locally:
   ```bash
   npm run dev
   ```
